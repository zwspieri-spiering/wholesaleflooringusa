# Wholesale Flooring USA Website

## Project Overview
This project is a multi-page HTML/CSS website created for CIS300. The site presents Wholesale Flooring USA services, products, and customer contact options.

## Live Links
- Repository: https://github.com/zwspieri-spiering/wholesaleflooringusa
- GitHub Pages: https://zwspieri-spiering.github.io/wholesaleflooringusa/

## Pages Included
- Home: index.html
- About Us: about-us.html
- Installation Services: installation-services.html
- Featured Flooring Products: featured-flooring-products.html

## Main Features
- HTML5 structure tags on each page (header, nav, main, footer)
- One external stylesheet (styles.css)
- Embedded HTML5 video on the home page
- Product comparison table on the Featured Flooring Products page
- Estimate request form with at least four fields
- External brand links in the Trusted Flooring Brands section
- Email text linked to on-site contact section (no mailto usage)
- Consistent navigation between all pages

## How to Run Locally
1. Open the project folder in VS Code.
2. Open index.html in a browser.
3. For best video playback, run with a local server and open:
   - http://localhost:8000/index.html

Example local server command (if Python is installed):

python -m http.server 8000
